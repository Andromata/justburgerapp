## Comenzando

Para clonar y levantar la aplicación será necesario preparar el escenario...


### Inicializar

1. Clona el Repositorio
   ```sh
   git clone https://github.com/Andromata/justburgerapp
   ```
2. Instala paquetes desde npm
   ```sh
   npm install
   ```
3. Ingresar a 
   ```sh
 http://localhost:3001
   ```
